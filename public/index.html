How it works

1. Share the room link with your guest
2. Both click Get Mic
3. Both click Start Recording
4. Talk — 3 recorders run simultaneously
5. Stop → files auto-download

*-hq.wav lossless master
*-meta.json VOX timestamps
*-whisper.webm 32kbps Opus

6. Send HQ + meta to peer via P2P
7. Stitch WAVs → stereo master
8. Open Transcriber → Whisper

Signal

Monitor: Opus (WebRTC, low-latency)
HQ record: WAV/PCM (local, lossless)
Whisper: 32kbps Opus webm × 2
  own mic + peer stream
Transfer: P2P data channel
